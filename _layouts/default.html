Welcome to Coding Fest

Coding Fest is a university-wide coding event organized by School of Computer Science, Faculty of Engineering to advocate coding culture to students. It showcases interesting coding projects, connects like-minded thinkers, and engages industry with students.

Coding Fest 2021 will be held in a hybrid mode on Wed 9th June.

Timeline

  • Application submission on Friday 2nd April. For project showcase applications, you only need to provide your team information and project description at this stage. You will have until the end of the semester to develop your project.
  • Showcase project poster submission on Friday 14th April. One page poster needs to be submitted from each showcase project.
  • Coding Fest 2021 day on Wed 9th June!
